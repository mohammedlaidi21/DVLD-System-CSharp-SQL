<h2> 🌟 Overview</h2>
DVLD is a comprehensive desktop application designed to manage the entire lifecycle of driving licenses. From the initial application and vision tests to license issuance, international permits, and the detain/release system. This project was built to simulate a real-world governmental system with strict business rules.

<h2>🏗️ System Architecture</h2>
<p>The project follows a strict <b>3-Layer Architecture</b> to ensure maintainability and scalability:</p>

<ul>
  <li><b>Presentation Layer (PL):</b> Handled via C# WinForms for a responsive User Interface.</li>
  <li><b>Business Logic Layer (BLL):</b> Contains all the validation and business rules (Applications, Licenses, etc.).</li>
  <li><b>Data Access Layer (DAL):</b> Manages all communication with the SQL Server database.</li>
</ul>

<h2>🛠️ Key Features</h2>
<h5>Application Management:</h5> Handle Local and International driving license applications.
<h5>Test Appointments:</h5>Management of Vision, Written, and Street tests.
<h5>License Lifecycle:</h5> Issuing, renewing, replacing (lost/damaged), and detaining licenses.
<h5>Driver & User Records:</h5> Full management system for people, drivers, and system users.

<h2>📊 Database Design</h2>
<p>The system relies on a complex relational database designed to handle high data integrity:</p>

<h5>Key Tables:</h5>
<ul>
  <li>People, Drivers, Users, Licenses, DetainedLicenses, and more.</li>
</ul>

<h5>Relationships:</h5>
<p>Structured with Foreign Keys to maintain data consistency across the entire system.</p>
